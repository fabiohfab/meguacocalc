

Instructions for multi-language package documentation (tutorials, etc):

* http://read-the-docs.readthedocs.io/en/latest/localization.html#localization-of-documentation

MEGUA has:

* pt documentation as the main project (megua.readthedocs.io)
* en documentation as a sub documentation project  (megua-english.readthedocs.io)

Starting in 

   megua.readthedocs.io

the user/reader can change language at the left bottom.
