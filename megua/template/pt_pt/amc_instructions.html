O software Auto Multiple Choice permite criar testes de escolha múltipla com variantes como questões abertas em que o docente marca a cotação ou questões em que várias opções estão corretas. No final os testes são digitalizados e o processo de marcação de respostas erradas ou corretas e soma de pontuação é automatizado com base no ficheiro digitalizado. Um ficheiro de cotações é construído. Eventualmente a correção do teste é enviada por email ao estudante.

Instruções breves:

  1. Verifique se o PDF está de acordo com o planificado ou selecione novas chaves ou exercícios. É normal que o ficheiro contenha marcas HTML que podem ser removidas ou transformadas.
  2. Abra um projeto novo usando o software AMC.
  3. Descarregue o ficheiro amc_test.tex e o ficheiro amcpt.sty para a pasta do projeto. (Talvez seja necessário mudar onome do ficheiro para groups.tex)
  4. Descarregue o zip com as imagens para essa mesma pasta e faça "unzip"
  5. Dentro do ficheiro amc_test.tex existe o comando \onecopy{3}{: modifique o 3 para o número de cópias com as questões baralhadas que pretende.
  6. Existe um grupomatematica dentro do ficheiro amc_test.tex. Se entender pode dividir esse grupo em outros grupos como por exemplo grupoalgebra, gruponumerica, etc, e modificar na parte final do ficheiro tex os comandos insertgroup e shufflegroup.
  7. Use o programa AMC para compilar e gerar um ficheiro final a imprimir e distribuir pelos alunos.
